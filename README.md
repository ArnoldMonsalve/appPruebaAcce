# Aplicación móvil híbrida desarrollada con **Ionic** y **Angular** para la gestión eficiente de tareas clasificadas por categorías.
## Requisitos instalados
* Node.js
* Ionic CLI
* Cordova
* Java JDK 17 y variables de entorno configuradas.
* Android Studio panda 3 con SDK Tools configurados.

Para Compilar y emular se realizo en telefono android MotoG84 con android 15.
