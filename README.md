# Aplicación móvil híbrida desarrollada con Ionic + Angular, diseñada para la gestión eficiente de tareas mediante clasificación categórica y control dinámico de funciones desde la nube.

## Entorno
* Frameworks: Ionic Framework (ultima versión) con Angular.
* Bridge Nativo: Capacitor & Cordova Plugins para acceso a hardware y servicios del sistema.
* Config: Firebase Remote Config para la gestión de Feature Flags en tiempo real.
* Almacenamiento de Datos: LocalStorage API, garantizando la disponibilidad de la información sin dependencia de una conexión constante a base de datos externa (Modo Offline).

## Requisitos
* Runtime: Node.js.
* CLI: Ionic CLI & Capacitor.
* Java: JDK 17 con variables de entorno configuradas.
* IDE: Android Studio Panda 3 (Electric Eel).

## Pruebas
* Dispositivo de prueba: Moto G84.
* Versión de Android: Android 15.
* Resultado: Compilación exitosa y verificación de persistencia de datos local tras el cierre de la aplicación.
